# 욕심쟁이 판다

## 풀이 방법

문제를 보자마자 dfs 혹은 bfs를 통해 문제를 해결하는 것임을 알았다. 그리고 또한, 무작정 완전탐색을 하는 것이 아닌 rem이라는 리스트를 별도로 만들어두어 더이상 가지 않아도 되는 길은 가지 않게하였다.

처음에는 bfs + memorization을 통해서 문제를 해결해볼려고 했으나 36%에서 시간초과가 발생하였다.

그래서 dfs + memorization을 통해 문제를 해결하니 통과하였다.

rem 리스트에는 해당 좌표에서 최대로 먹을 수 있는 대나무의 개수를 저장하였다. 그리고 rem[i][j] == 0 이라는 말은 해당 좌표의 대나무를 검사하지 않았으므로 dfs를 통해 파악한다.
