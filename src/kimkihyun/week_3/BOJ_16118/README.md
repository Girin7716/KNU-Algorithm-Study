# 달빛 여우

## 풀이 방법

이 문제에서는 여우와 늑대의 최단거리를 `다익스트라`를 통해 거리를 구해서 각 노드의 최단거리 리스트인 distance를 여우와 늑대를 비교해서 여우>늑대일 경우 result+=1를 해서 출력하면 된다.

여우의 경우 기본적인 다익스트라 알고리즘을 이용하여 distance를 구하면 된다.

하지만, 늑대의 경우는 이동속도가 처음에는 2배, 나중에는 1/2배이므로 이를 다익스트라로 구하기위해서는, 우선순위큐에 원소를 넣을때 다음과 같이 넣어준다.
```python
heapq.heappush(pq, (0, 1, 1))  # dist, node, index
```
여기서 index의 의미는 index == 0 : 다음 속력은 절반, index ==1 : 다음 속력은 2배를 의미한다 그리하여 wolf의 경우 각 노드의 distance에는 2개의 값이 저장되며 이 값은 index의 기준으로 값이 바뀐다.

위 두가지 과정을 거쳐 여우와 늑대의 distance 리스트를 비교해서 값을 출력해주면 된다.
