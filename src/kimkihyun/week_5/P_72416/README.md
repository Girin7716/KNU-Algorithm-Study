# 매출 하락 최소화

## 풀이방법
tree 구조와 dp를 사용하여 문제를 해결해야한다.

dp[n][0]의 의미는 n번 노드가 참석하지 않았을때의 매출 손실액.<br>
dp[n][1]의 의미는 n번 노드가 참석했을때의 매출 손실액.

여기서 단말 노드(leaf node, 자식이 없는 노드)를 제외한 노드들은 자식노드일 수도, 부모 노드 일 수도 있다. 그래서 leaf node 바로 위 부모 노드는 팀장이 된다.

팀장 노드 dp는 3가지가 나온다.<br>
1. 팀장이 참석하는게 이득
2. 팀장이 참석하는게 이득이 아니지만 그 자식 노드 중 하나라도 참석하는게 무조건 이득
3. 팀장 노드 참석하지 않고 자식 노드들 모두 참석하지 않는게 이득.
