BFS를 이용한 문제이다.
Queue를 이용해서 구현했다.

for문으로 육지일때 BFS를 실행해서
보물이 묻혀있는 곳의 최단 거리로 이동하는 시간을 찾았다.

BFS 함수에서는 지금 현재 위치를 큐에 넣고 사방으로 육지거나, 가지 않았던 곳이구나, 갈 수 없는 곳이 아닐 경우 큐에 넣는다.
(continue로 했음)
큐에서 한 위치씩 꺼내서 앞의 과정을 큐가 빌때까지 반복한다.
큐가 비었을 때 거리를 리턴하면 보물 간의 최단 거리 이동시간을 찾을 수 있다.
