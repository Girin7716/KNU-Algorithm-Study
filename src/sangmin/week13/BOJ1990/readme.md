1. 코드 설명

에라토스테네스의 체를 이용하여 모든 소수를 구해주었다.

여기서 중요한점은 문제에서 주어진 구해야하는 소수의 범위는 max 100,000,000 까지 이다.

하지만 실질적으로 거꾸로해도 같은 수는 10,000,000 이상으로 나오지 않는다.

이를 이용하여 에라토스테네스의 체 범위를 10,000,000 까지로 해주고 해당 숫자를 문자열로 바꾸어 앞뒤가 같은지 판단해준다.
